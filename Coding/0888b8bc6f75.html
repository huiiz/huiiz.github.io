Python第三方库yaml

发表于2023-03-14|更新于2023-10-05|Coding
|阅读量:

介绍

YAML 全称 YAML Ain't Markup Language，它是一种以数据为中心的标记语言，比 XML 和 JSON 更适合作为配置文件[ref][YAML教程（快速入门版） (biancheng.net)](http://c.biancheng.net/spring_boot/yaml.html)[/ref]。

PyYAML是Python中用于处理YAML的第三方库。

安装

PyYAML 6.0要求 Python >=3.6[ref][PyYAML · PyPI](https://pypi.org/project/PyYAML/)[/ref]。

YAML文件

后缀

xxx.yml 或 xxx.yaml

语法

树形结构，相同的前缀只需要写一次

  • 使用 空格缩进 来表示层级关系，空格数不重要，相同层级左对齐即可。不能用tab
  • 使用key: value，:后面要加一个空格（重要）
  • 大小写敏感
  • #表示注释
